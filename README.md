# vue-preset-template
vue-cli 项目模板, 开发语言为TypeScript, 默认支持sass

## 如何使用
安装全局的vue-cli

    npm install -g @vue/cli

使用vue-cli命令创建项目

    vue create --preset sjhleo/vue-preset-template projectName
